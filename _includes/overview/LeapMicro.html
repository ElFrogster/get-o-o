Leap Micro is the Host OS that you were looking for.

Leap Micro is an ultra-reliable, lightweight operating system purpose-built for containerized and virtualized workloads. It is based on SUSE Linux Enterprise Micro which leverages the enterprise hardened security and compliance components of SUSE Linux Enterprise and merges them with a modern, immutable, developer-friendly OS platform.

Users who want to try Leap Micro in VM running on either Xen or KVM should use the new self-install image which boots and extracts included .raw.xz image to the drive and reboots. Users with Raspberry PI and other SOCs should use directly our preconfigured image together with ignition and combustion. Both preconfigured and self-install images are intended to be used with a Ignition and Combustion enabled USB drive to set a password and further configuration on the first boot.
